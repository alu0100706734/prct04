¿Qué es rvm (Ruby Version Manager )?

RVM es una plataforma software designada para gestionar multiples instalaciones de ruby en un mismo dispositivo

¿Cu´ales son las diferencias entre la ejecuci´on del programa "Hola Mundo" con el int´erprete
de Ruby (ruby helloWorld.rb) y el int´erprete interactivo del ejercicio anterior?


La diferencia entre el intérprete de Ruby y el intérprete interactivo es que, con Ruby solo se intérpreta el código y muestra el resultado, mientras que con el intérprete interactivo muestra la línea, la sentencia, lo que representa esa sentencia y además lo que devuelve.
